# 📄 PDF 分类工具使用说明

## 🎯 功能简介
快速浏览 PDF 文件首页并分类归档，支持键盘快捷键操作，提高分类效率。

---

## ⌨️ 快捷键

| 按键 | 功能 |
|:---:|:---|
| **1** | 归类到"好" - 复制当前 PDF 到目标目录 |
| **2** | 跳过 - 不处理当前 PDF，进入下一个 |

---

## 🔧 操作按钮

- **✅ 归类到好 (1)** - 将 PDF 复制到 `非常好` 子目录
- **➡️ 跳过 (2)** - 跳过当前文件
- **🔄 重开当前目录** - 重置当前目录从头开始
- **⬅️ 重开上一目录** - 返回上一个目录重新开始

---

## 📊 顶部工具栏

- **🔄 加载任务** - 扫描 `files_debug` 目录加载任务
- **🔁 重新开始全部** - 重置所有任务进度

---

## 🧭 历史记录与回溯

- 左侧 **📊 历史记录** 会实时显示操作记录（最新优先）
- 点击任意历史记录可回溯到对应时间点
- 回溯后主界面会自动恢复到当时的目录与进度

---

## 💡 使用技巧

1. 使用 **数字键 1/2** 快速分类，无需点击按钮
2. 查看右侧 **统计** 了解当前目录和全局进度
3. 误操作可使用 **重开当前目录** 重新处理
